# Deepak Bajaj Portfolio

A professional, visually stunning portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Dark-themed, clean, and professional UI.
- **3D Elements**: Interactive 3D background and tilt effects using Three.js and Framer Motion.
- **Animations**: Smooth transitions and interactions.
- **Responsive**: Fully responsive design for all devices.
- **Sections**:
  - Hero
  - About
  - Projects
  - Skills
  - Contact

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- Update content in `src/components/sections/` files.
- Modify styles in `src/app/globals.css`.
- Update metadata in `src/app/layout.tsx`.
